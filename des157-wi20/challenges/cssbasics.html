Inheritance in CSS

Inheritance in CSS is when the properties of a child element inherits the properties of the parent element. One example is when the color of the header font(child element) inherits the color of the body font(parent element). This happens when the color of the header font is not defined. Other elements that can inherit the properties of the body are the paragraphs. If the paragraph font color is defined, it won't inherit the font color of the body.

Specificity in CSS

Specificity is how the browsers determine the importance of the CSS Styles. In other words, which specific style declaration is applied to a certain element. A compound selector can customize the content that you want to style using more than one class, ID, and/or tag. Overall, it has a greater priority compard to other selectors. The multiple selector allows you to group various elements together with the same set of styles.

ID and Class Attributes

An ID is a selector that is used to identify one element using the hash character. Class is a selector that is used to identify more than one element using the period character.
