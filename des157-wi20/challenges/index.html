Why we use the Normalize CSS Script

We use the normalize.css script to normalize the content of the code throughout various browsers. This applies to both browsers on mobile and desktop. Overall, this improves the functionality and usability of the code.

What is Semantic Markup

A semantic markup creates meaning of the elements on the web page and helps identify those various elements. It acts as a skeleton for the web page as it makes it easier to organize and group things together.
