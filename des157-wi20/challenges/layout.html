Structure, Phrasing and Display

Structural elements take up the space of the entire parent container such as div and paragraph. These elements start on a new line. While the phrasing elements occupy the space that is bounded by its tag. These elements do not cause a line break. Display properties control the layout of the page as it specifies how an element is displayed. The different properties are: inline, block, and none. Display block changes the layout of the the block elements such as the div container. While the inline block changes the layout of the phrasing elements such as paragraphs.

Box Model

The CSS Box model describes how the elements are layed out on the browsers, along with their corresponding dimensions. It affects width, padding, margin, and borders in terms of where they are located on the page and how much space they take up. The box model can be changed with the box-sizing property in CSS by defining how the width and height of a certain element are measured through padding and border.

Background Images

The images on the page inside tags are affected by the properties of the element it corresponds too. They are independent from the text of the page. While the images loaded through CSS as background images take up the space of the section or element is corresponds to. The text and background colors within the images tag can be layered above the images.
